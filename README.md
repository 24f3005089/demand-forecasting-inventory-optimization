# Demand Forecasting & Inventory Optimization

## Overview

This project develops a machine learning-based demand forecasting system for retail businesses.

Using historical sales, inventory, pricing, promotions, weather conditions, and seasonal factors, the model predicts product demand and provides inventory optimization recommendations.

## Dataset

* 76,000 records
* Retail demand forecasting dataset
* Features include:

  * Inventory Level
  * Units Sold
  * Units Ordered
  * Price
  * Discount
  * Promotion
  * Competitor Pricing
  * Weather Conditions
  * Seasonality

## Machine Learning Models

* Random Forest Regressor
* XGBoost Regressor

## Results

| Metric   | Value |
| -------- | ----- |
| R² Score | 0.92  |
| MAE      | 9.97  |
| RMSE     | 13.28 |

## Key Insights

* Promotions significantly influence demand.
* Seasonal factors affect purchasing patterns.
* Pricing and competitor pricing impact sales.
* Inventory optimization can reduce stockouts and overstock situations.
